## Käynnistysohjeet
Lataa projektin riippuvuudet komennolla poetry install

Siirry virtuaaliympäristöön ja aja index.py komentorivissä src kansiosta

Windowsilla tämä toimii komennolla py index.py

Avaa selaimessa local-host server http://127.0.0.1:5001
